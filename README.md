# Nuxt Firebase Todo List

[Live Demo](https://miksin.github.io/nuxt-firebase-todo-list)

Todo List Web App built with [Nuxt.js](https://nuxtjs.org/) and Firebase

## Features

- [x] Todo List
  - [x] Basics (CRUD)
  - [x] Mobile support
- [x] Auth
  - [x] Middleware
  - [x] Firebase support
